# PARKING MANAGEMENT SYSTEM

## System Description
ParkEase is a comprehensive Parking Management System designed to solve the common challenges faced by parking facilities—manual tracking, billing errors, slot congestion, and inefficient space utilization.

The system provides a real-time dashboard, full CRUD management for all entities, automated slot status updates, and precise fee calculation based on parking duration and slot type.

---

## Key Features

- **Real-time Dashboard** — Live slot availability, active tickets, revenue stats, and recent activity  
- **Customer Management** — Full CRUD with vehicle count tracking  
- **Vehicle Management** — Register and manage vehicles linked to customers  
- **Smart Slot Management** — Visual grid view, auto-status update when tickets are issued/completed  
- **Automated Ticketing** — Issue tickets for 1 or 2 vehicles on Premium slots; automatic slot occupancy  
- **Accurate Billing** — Duration calculated on checkout; billed in 15-minute increments (minimum 1 hour)  
- **Payment Processing** — Cash, Credit Card, Debit Card, GCash, PayMaya; full receipt view  
- **Auto Slot Release** — Slot status automatically reverts to Available after checkout  
- **Search & Filter** — Search and filter across all data tables  
- **Responsive Design** — Works on desktop, tablet, and mobile
